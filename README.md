# Internet Monitor

Simple DevOps tool for monitoring internet connectivity.

## Features

- Checks internet connection periodically
- Logs connection status
- Detects outages
- Lightweight monitoring tool

## Technologies

- Python
- Logging
- Ping

## Usage

Run the monitor:

python monitor.py

## Configuration

Edit `config.py`:

HOST = "1.1.1.1"
CHECK_INTERVAL = 30
